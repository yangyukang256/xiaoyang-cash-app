# 小羊记账 — 产品文档与项目规则

> 项目概述：一款面向个人用户的记账APP，用于记录日常生活中每一笔花销（人民币），支持2级分类管理。

## 第二部分：产品需求文档（PRD）

### 2.1 产品概述
| 项目 | 内容 |
|------|------|
| 应用名称 | 小羊记账 |
| 目标用户 | 个人用户（中国境内，使用人民币） |
| 核心功能 | 记录每一笔花销，支持2级分类管理 |
| 语言 | 简体中文 |
| 货币 | 人民币（CNY/¥） |

### 2.2 功能需求

#### 2.2.1 记账功能（核心）
| 功能 | 说明 |
|------|------|
| 新增支出 | 用户可添加一笔支出记录 |
| 支出金额 | 输入人民币金额，支持到分（0.01元） |
| 记账时间 | 默认为当前时间，也可手动选择日期（精确到天） |
| 备注说明 | 可选填，用于补充说明这笔花销的具体用途 |
| 支出分类 | 必须选择一个2级分类（一级大类 → 二级小类） |
| 查看记录 | 按时间倒序展示所有记账记录 |
| 编辑记录 | 支持修改已记的支出 |
| 删除记录 | 支持删除某笔支出 |

#### 2.2.2 数据查看功能
| 功能 | 说明 |
|------|------|
| 总支出统计 | 查看指定时间段的总支出 |
| 分类统计 | 按一级大类/二级小类查看支出分布 |
| 月度汇总 | 按月查看支出汇总 |

| 🏠 居住生活 | 房租、水电燃气、物业费、网费、家居日用品、维修 |
| 👕 购物消费 | 服装鞋帽、数码产品、日用品、美妆护肤、礼物、宠物 |
| 🎮 休闲娱乐 | 电影演出、游戏充值、运动健身、旅游出行、聚会聚餐 |
| 🏥 医疗健康 | 看病挂号、药品、体检、保健品 |
| 📚 教育学习 | 课程培训、书籍购买、文具用品、考证报名 |
| 📱 通讯通信 | 话费充值、流量套餐 |
| 💰 金融理财 | 信用卡还款、借贷支出、保险费用 |
| 📦 其他 | 红包支出、捐赠、快递费、其他 |

> **说明**：以上分类如用户觉得不合适，可以随时调整。分类是项目的基础数据结构，后续开发中可能会需要用户确认或修改。

**这是啥？**
做一个网页版的记账本，电脑浏览器打开就能用。数据存在你自己电脑浏览器里（localStorage）。
不需要服务器，不需要买域名，不需要部署。用 GitHub Pages 可以直接免费发布到网上，手机也能打开。

**开发技术：** HTML + CSS + JavaScript (Vanilla JS) 或 Vue.js

**✅ 优点：**
- 最简单、开发最快（几天就能做完）
- 完全免费，不需要买服务器、域名
- 不用注册账号，打开即用
- 国内镜像都能用，不需要科学上网

**❌ 缺点：**
- 数据存在浏览器本地，清除浏览器缓存数据会丢失（但可以加导出功能备份）
- 不能多设备同步（手机上记的账电脑上看不到）
- 功能相对简单，不适合复杂统计

**💰 成本：** 0 元（纯免费）

### 方案二：📱 微信小程序

**这是啥？**
就是你在微信里搜到的那种"不用安装、点开即用"的小程序。对于中国用户来说，不用额外安装APP。通过微信小程序搜索"小羊记账"就能打开。

**开发技术：** 微信小程序原生框架 / Taro / UniApp

**✅ 优点：**
- 微信生态内，用户不用安装，打开即用
- 可使用微信云开发，数据存在腾讯服务器上，不会丢失
- 可以发布分享给朋友用
- 手机电脑都能用（微信有PC版）

**❌ 缺点：**
- 需要注册微信小程序开发者账号（个人版 30元/年认证费）
- 需要经过微信审核才能发布上线
- 开发受微信平台限制（不能随意调用手机功能）
- 国内镜像资源较少，部分需要直接访问微信官方文档

**💰 成本：** 约 30 元/年（微信认证费）+ 可能需购买云开发资源（几十元/月）

### 方案三：🌐 前后端分离 Web 应用

**这是啥？**
做一个真正的网站应用，有后端服务器保存数据，有前端页面展示。数据存在服务器数据库里，无论换手机还是换电脑，登录账号就能看到所有数据。类比"在线记账本"。

**开发技术：**
- 前端：React / Vue.js
- 后端：Node.js (Express) / Java (Spring Boot) / Python (Flask)
- 数据库：MySQL / SQLite / MongoDB
- 部署：服务器（需要购买）

**✅ 优点：**
- 数据在云端，多设备同步，不怕丢失
- 功能最完整，可以做复杂的统计图表、数据分析
- 可以设置登录账号，多人使用
- 未来可以扩展成手机APP（套壳Web或开发API给移动端）

**❌ 缺点：**
- 需要购买云服务器（最低配置几十元/月）
- 开发周期较长（前后端都要开发）
- 需要部署和运维（虽然个人项目运维量很小）
- 需要注册域名（几十元/年）

**💰 成本：** 约 50~100 元/月（服务器+域名，学生机或轻量服务器更便宜）

### 方案四：🦋 跨平台移动APP（Flutter）

**这是啥？**
用一套代码同时做出 iOS 和 Android 两个手机APP，像真正的APP一样可以安装到手机上使用。数据可以存在手机本地，也可以连接云端服务器。

**开发技术：** Flutter (Dart语言) + SQLite（本地存储）

**✅ 优点：**
- 真正的手机APP，体验最好
- iOS 和 Android 都支持，一套代码搞定
- 界面可以做得非常精致
- 支持本地存储+云端存储双模式

**❌ 缺点：**
- 开发技术相对复杂，Dart语言国内资料不如JS丰富
- 需要安装 Android Studio 等开发工具（较大，几个GB）
- 如果要发布到 App Store，需要 99美元/年的苹果开发者账号
- 部分依赖包可能需访问国外资源（但大多有国内镜像）

**💰 成本：**
- Android 发布：约 25 美元一次性（Google Play）
- iOS 发布：99 美元/年（Apple Developer）
- 或选择只发 Android / 只做本地APK安装：0元
- 开发工具免费
