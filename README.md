## 📖 项目简介

**小羊记账**是一款纯前端记账应用，旨在帮助个人用户轻松记录日常生活中的每一笔收支。所有数据均存储在浏览器本地，无需注册账号，打开即用。

- 💸 **记支出** — 记录每一笔花销，支持 10 个一级大类、40+ 个二级小类
- 💰 **记收入** — 支持工资、副业、理财等 5 个收入大类
- 📊 **看统计** — 月度汇总、分类占比一目了然
- 🔒 **数据本地** — 存在你自己浏览器里，隐私安全

---

## ✨ 功能特性

### 📝 记账功能

| 功能 | 说明 |
|------|------|
| 💸 记一笔支出 | 选择分类（2级）+ 输入金额 + 可选备注 |
| 💰 记一笔收入 | 支持工资、副业、理财等收入类型 |
| 📅 选择日期 | 默认当天，可手动修改（精确到天） |
| ✏️ 编辑记录 | 随时修改已记的收支信息 |
| 🗑️ 删除记录 | 一键删除不需要的记录 |

### 📃 账单查看

- **账单列表** — 按时间倒序展示所有记录，支出红色（-¥）、收入绿色（+¥）
- **筛选过滤** — 支持「全部 / 支出 / 收入」三种视图切换
- **分类展示** — 每条记录显示一级分类图标 + 二级分类名称
- **备注显示** — 可选填备注，补充说明每笔花销的用途

### 📊 数据统计

- **月度总览** — 查看指定月份的支出、收入、结余
- **分类统计** — 按一级大类查看支出/收入分布，带进度条占比
- **二级明细** — 展开每个大类，查看二级小类的详细金额
- **月份切换** — 支持切换查看不同月份的数据

### 💾 数据管理

- **自动保存** — 每笔记录自动存入浏览器本地，无需手动保存
- **导出 CSV** — 一键导出为 Excel/WPS 可打开的 CSV 文件
- **导入备份** — 支持导入 JSON 备份文件恢复数据

## 🚀 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) >= 18（建议使用 LTS 版本）
-  npm（随 Node.js 一起安装）

### 安装与运行

```bash
# 1. 克隆仓库
git clone https://github.com/your-username/xiaoyang-cash-app.git
cd xiaoyang-cash-app

# 2. 进入项目目录
cd xiaoyang-cash-app

# 3. 安装依赖
npm install --registry=https://registry.npmmirror.com

# 4. 启动开发服务器
npm run dev

# 5. 在浏览器中打开
# 访问 http://localhost:5173
```

### 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录中，可直接部署到任何静态文件服务器。

## 🌐 部署方式

### GitHub Pages（免费推荐）

1. 构建项目：`npm run build`
2. 将 `dist/` 目录内容推送到 `gh-pages` 分支
3. 在仓库 Settings → Pages 中启用 GitHub Pages

### 单文件 HTML

项目根目录下的 `xiaoyang-cash-app.html` 是打包好的单文件版本，双击即可打开使用，无需启动任何服务器。
